# **App Name**: ScentTrack Delivery

## Core Features:

- Order Management: Add, edit, and manage perfume orders with details such as customer name, address, product details, and payment type.
- Shipping Label Generation: Automatically generate PDF shipping labels with order details and a unique QR code containing the Order ID.
- QR Code Generation: Generate dynamic QR codes for each order, encoding the Order ID for quick access via scanning.
- Delivery Scanner Module: Enable delivery personnel to scan QR codes, fetch customer details, and trigger OTP delivery verification.
- OTP Verification: Send OTPs via SMS to customers upon scanning, verify the OTP entered by the delivery agent, and confirm the order delivery.
- Order Status Updates: Update the order status to 'Shipped,' 'Out for Delivery,' or 'Delivered (OTP verified)' and track delivery date/time.
- Delivery Insights Tool: AI tool uses delivery data and location (when available) to infer whether any information should be flagged for closer human review.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), inspired by perfume branding, to convey luxury and sophistication.
- Background color: Light lavender (#E1DCEF), a desaturated hue of the primary color for a gentle contrast.
- Accent color: Soft blue (#3F51B5), an analogous hue that provides a cool contrast and enhances readability.
- Font: 'Inter', a sans-serif font for a modern and clean user interface; appropriate for both headings and body text.
- Use elegant, minimalist icons for order statuses and delivery actions to maintain a professional appearance.
- A clean, card-based layout to showcase order information with easy-to-access actions.
- Subtle transition animations and loading indicators to enhance user experience during QR code scanning and status updates.
